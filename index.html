Daily review budgets

Independent queues: filling one budget never blocks due cards from the other. New bulk-uploaded cards are introduced adaptively within spare budget.

Hourly micro-review

At most one buzz per hour inside these windows, and only when a recall card is actually due — otherwise the watch never wakes.

Scheduling algorithm

FSRS schedules each card from a fitted memory model (difficulty · stability · retrievability) so intervals target these retention probabilities directly. Lower retention = fewer, longer reviews. Existing SM-2 state migrates automatically on the next rating. The SM-2 card applies only when the legacy scheduler is selected.
